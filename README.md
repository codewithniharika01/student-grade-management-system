# 🎓 Student Grade Management System

A console-based Student Grade Management System built using Python. This application helps manage student records by allowing users to add, update, delete, and view student grades.

## Features

* Add new student records
* Update existing student grades
* Delete student records
* Display all students and grades
* Menu-driven interface
* Input validation and error handling

## Technologies Used

* Python

## Concepts Used

* Dictionaries
* Functions
* Loops
* Conditional Statements
* Exception Handling
* CRUD Operations

## Project Structure

Student Grade Management System
│
├── Add Student
├── Update Grade
├── Delete Student
├── View Students
└── Exit Program

## How to Run

1. Clone the repository

```bash
git clone <repository-link>
```

2. Navigate to the project directory

```bash
cd student-grade-management-system
```

3. Run the Python file

```bash
python student_grade_management.py
```
